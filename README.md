## Setup
```bash
git clone --recurse-submodules git@github.com:byang12159autel/crisp_framework.git
```
Switch all repos to branch
```bash
git switch ben/feature/mink_integration 
```

The submodules of the following repositories should be cloned
```bash
git clone git@github.com:byang12159autel/crisp_controllers_demos.git
git clone git@github.com:byang12159autel/crisp_py.git
git clone git@github.com:byang12159autel/crisp_gym.git
git clone git@github.com:byang12159autel/crisp_controllers.git
```
